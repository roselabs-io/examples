Sage Live Chat Widget

Embeddable AI-powered live chat for your website

Try it now! Click the chat bubble in the bottom-right corner to start a conversation.

Quick Start

Add this script tag to your website to enable the chat widget:

<script
  src="https://cdn.roselabs.io/sage-chat.min.js"
  data-sage-api-key="sk_sage_..."
></script>

Configuration Options

Customize the widget with data attributes:

<script
  src="https://cdn.roselabs.io/sage-chat.min.js"
  data-sage-api-key="sk_sage_..."
  data-sage-color="#6366f1"
  data-sage-position="bottom-right"
></script>

ES Module Usage

For React, Vue, or other frameworks:

import { SageChat } from '@roselabs-io/sage/chat';

// Initialize the chat widget
const chat = SageChat.init({
  apiKey: 'sk_sage_...',
  primaryColor: '#6366f1',
  visitor: {
    email: 'user@example.com',
    name: 'John Doe',
  },
});

// Control the widget programmatically
chat.open();
chat.send('Hello!');
chat.identify({ email: 'new@email.com' });
chat.close();
chat.destroy();
